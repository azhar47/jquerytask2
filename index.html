What is Java ?

Java is a programming language and a platform. Java is a high level, robust, secured and object-oriented programming language.

What is C ?

C programming is considered as the base for other programming languages, that is why it is known as mother language.

What is JavaScript ?

JavaScript is a high-level, dynamic, untyped, and interpreted programming language. It has been standardized in the ECMAScript language specification.

What is PHP ?

PHP is a server-side scripting language designed primarily for web development but also used as a general-purpose programming language. Originally created by Rasmus Lerdorf in 1994, the PHP reference implementation is now produced by The PHP Development Team.

What is Front-end Web development?

Front-end web development, also known as client-side development is the practice of producing HTML, CSS and JavaScript for a website or Web Application so that a user can see and interact with them directly.
